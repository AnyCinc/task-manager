# コピペ用 関数リファレンス（手動構築する場合）

Apps Scriptを使わず手動で構築したい場合のチートシートです。

---

## 🔷 01_データ抽出 シート

### A1セル（IMPORTRANGE + QUERY）

元データ：A列=日付、H列=営業担当者、I列=種類 を想定。

```
=QUERY(
  IMPORTRANGE("元スプレッドシートのURL","受電報告!A:I"),
  "SELECT Col1, Col8, Col9 WHERE Col1 IS NOT NULL LABEL Col1 '日付', Col8 '営業担当者', Col9 '種類'",
  1
)
```

初回はセル横の「アクセスを許可」をクリック。

## 🟢 02_分析ダッシュボード シート

### 日付範囲セル
| セル | 内容 |
|---|---|
| B3 | 開始日（データ検証：日付） |
| D3 | 終了日（データ検証：日付） |

### A6（担当者一覧を自動生成）
```
=SORT(UNIQUE(FILTER('01_データ抽出'!B2:B, '01_データ抽出'!B2:B<>"")))
```

### B6（受電数）
```
=IF($A6="","",COUNTIFS(
  '01_データ抽出'!$A:$A, ">="&$B$3,
  '01_データ抽出'!$A:$A, "<="&$D$3,
  '01_データ抽出'!$B:$B, $A6
))
```

### C6（案件化数）
```
=IF($A6="","",COUNTIFS(
  '01_データ抽出'!$A:$A, ">="&$B$3,
  '01_データ抽出'!$A:$A, "<="&$D$3,
  '01_データ抽出'!$B:$B, $A6,
  '01_データ抽出'!$C:$C, "案件化"
))
```

### D6（案件化率）
```
=IF(OR($A6="",B6=0),"",C6/B6)
```
セル書式：**表示形式 → パーセント → 小数第1位**

B6〜D6を下方向に200行ほどコピー（担当者追加に備える）。

## 🟧 合計行（例：208行目）
```
A208:  合計
B208:  =SUM(B6:B205)
C208:  =SUM(C6:C205)
D208:  =IFERROR(C208/B208,0)
```

---

## 🎨 条件付き書式（D列）

1. D6:D205 を選択
2. 表示形式 → 条件付き書式 → カラースケール
3. 最小値=赤 / 中間=黄 / 最大値=緑

---

## 📊 グラフ設定

| グラフ種 | データ範囲 | 配置 |
|---|---|---|
| 横棒グラフ（受電数ランキング） | A5:B205 | F6付近 |
| 縦棒グラフ（受電数 vs 案件化数） | A5:C205 | F24付近 |
| 折れ線（案件化率） | A5:A205 + D5:D205 | F42付近 |
