# 営業分析スプレッドシート セットアップガイド

元データ「(2024年3月〜 日本人スタッフ)」の `受電報告` タブから、
営業担当者ごとの **受電数 / 案件化数 / 案件化率** を日付範囲指定で自動集計する
分析スプレッドシートを構築します。

## 📁 ファイル構成

| ファイル | 用途 |
|---|---|
| `README.md` | 本手順書 |
| `setup.gs` | Google Apps Script：シート構成・関数・書式を一括自動生成 |
| `formulas.md` | コピペ用の関数リファレンス（手動構築する場合） |

---

## 🚀 セットアップ手順（推奨：Apps Scriptで自動構築）

### 1. 新規スプレッドシートを作成
Google Driveで新規スプレッドシートを作成し、任意の名前を付けます（例：`営業分析_日本人スタッフ`）。

### 2. Apps Scriptエディタを開く
メニュー `拡張機能 → Apps Script` をクリック。

### 3. スクリプトを貼り付け
`setup.gs` の内容を全てコピーして、`コード.gs` に貼り付け、保存（Ctrl+S）。

### 4. 元データのURLを設定
スクリプト冒頭の `SOURCE_URL` に、元スプレッドシートのURLを貼り付け。

```javascript
const SOURCE_URL = 'https://docs.google.com/spreadsheets/d/xxxxxxxxxxxxxxxxxx/edit';
```

### 5. `buildAll()` を実行
Apps Scriptの関数選択プルダウンで `buildAll` を選び、`▶ 実行` をクリック。
初回のみ権限承認が求められます（「詳細」→「安全でないページに移動」→「許可」）。

### 6. IMPORTRANGEの接続承認
スプレッドシートに戻ると、`01_データ抽出` シートのA1セルに `#REF!` が表示されます。
セルをクリックして **「アクセスを許可」** をクリック。

完了！🎉 `02_分析ダッシュボード` でB3（開始日）とD3（終了日）を入力すれば自動集計されます。

---

## 🗂️ 出来上がるシート構成

### `01_データ抽出`（青タブ・保護）
IMPORTRANGE + QUERYで元データから **日付 / 営業担当者 / 種類** の3列だけを抽出。

### `02_分析ダッシュボード`（緑タブ・利用者が操作）
```
 A              B            C            D
1 【営業分析ダッシュボード】
2
3 開始日:       2024/03/01   終了日:      2024/12/31
4
5 営業担当者     受電数       案件化数     案件化率
6 (UNIQUE自動)   (COUNTIFS)   (COUNTIFS)   (IFERROR)
...
```
下部に合計行と、受電数ランキング・受電vs案件化の内訳・案件化率の3グラフが自動配置されます。

### `03_設定`（グレータブ・管理者用）
元スプレッドシートのURLや対象列の設定を保管。

## 🛠️ 元データ列の位置が異なる場合

デフォルトでは以下を想定しています：

| 列 | 内容 |
|---|---|
| A列 | 日付 |
| H列 | 営業担当者 |
| I列 | 種類 |

列位置が異なる場合は `setup.gs` 冒頭の以下定数を変更してください：

```javascript
const COL_DATE  = 1;  // A列
const COL_REP   = 8;  // H列
const COL_TYPE  = 9;  // I列
```
